### Linux

```shell
curl -L https://github.com/jenkins-x-labs/jx-labs/releases/download/v{{.Version}}/jx-labs-linux-amd64.tar.gz | tar xzv 
sudo mv jx-labs /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x-labs/jx-labs/releases/download/v{{.Version}}/jx-labs-darwin-amd64.tar.gz | tar xzv
sudo mv jx-labs /usr/local/bin
```
